# 产品上新与下架会审工具

这是由原“南京德基新店开业总控驾驶舱”替换升级而来的产品上下架协作工具。

## 免费线上架构

- GitHub Pages：提供公网页面
- Supabase：保存团队共享数据
- GitHub Actions：`main` 分支更新后自动发布

## 当前能力

- 热厨、水吧、烘焙分别维护多道上新与下架菜品
- 菜品名称、上下架时间、会审部门、执行意见可编辑
- 研发、市场、营运、厨政、采购会审意见可编辑
- 新增与删除上下架菜品
- 云端自动保存与每 30 秒同步
- 一键导出 PDF
- 品牌 VI 蓝色与橙色视觉

## 云端数据

工具复用原 Supabase 项目和 `opening_tasks` 表，不需要新建数据表。

上下架工具的完整共享状态保存在以下专用记录中：

```text
id = product-change-dashboard-state-v1
```

原来的开业任务数据不会被删除，新页面只读取上下架工具的专用记录。

## 发布

推送到 `main` 分支后，`.github/workflows/pages.yml` 会自动部署 GitHub Pages。
